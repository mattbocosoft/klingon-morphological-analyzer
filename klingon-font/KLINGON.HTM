Date of last change: December 14th, 1999

General Information

This font was created by Lars Kasper based on official symbols from the Star Trek TV shows and movies and symbols presented at the Klingon Language Institute [1].

Installation Notes

Install the font into your Windows system or any other systems that can handle True Type Fonts (TTF). Put the KLINGON.TTF-file in a directory on your system, e.g. C:\Windows\Fonts. In Windows you can manage the fonts in the System Control Panel/Fonts.
